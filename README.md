# bouzuya icon generator

bouzuya icon (`([:-P`) generator. 

## Installation

    $ npm install -g bouzuya/bouzuya-icon-generator

## Usage

    $ big
    $ ls -1
    bouzuya-120x120.png
    $ big -s 480
    $ ls -1
    bouzuya-120x120.png
    bouzuya-480x480.png
